R Package Reference

This appendix lists the key R packages used throughout the book, organized by purpose.

Core PK/PD Packages

  • rxode2 — ODE-based PK/PD simulation and modeling engine
  • nlmixr2 — Nonlinear mixed-effects modeling (population PK/PD) with FOCEi and SAEM
  • mrgsolve — Fast ODE-based PK/PD simulation in C++
  • PKNCA — Noncompartmental analysis (AUC, Cmax, t½, etc.)
  • PK — Basic pharmacokinetic analysis functions

Statistical Modeling

  • nlme — Linear and nonlinear mixed-effects models
  • lme4 — Linear mixed-effects models
  • saemix — SAEM algorithm for nonlinear mixed-effects models
  • brms — Bayesian regression models using Stan
  • cmdstanr — Lightweight R interface to CmdStan

Data Manipulation and Visualization

  • ggplot2 — Grammar of graphics for publication-quality plots
  • dplyr — Data manipulation (filter, mutate, summarize)
  • tidyr — Tidy data tools (pivot, nest, fill)
  • readr — Fast CSV/file reading
  • readxl — Read Excel files
  • patchwork — Combine multiple ggplot2 plots
  • ggrepel — Non-overlapping text labels in plots

Reporting and Reproducibility

  • knitr — Dynamic report generation
  • rmarkdown — R Markdown documents
  • quarto — Next-generation publishing system

Survival and Clinical Endpoints

  • survival — Survival analysis (Kaplan-Meier, Cox PH)
  • survminer — Visualization of survival data

Differential Equation Solvers

  • deSolve — General ODE/PDE/DDE solver
  • minpack.lm — Levenberg-Marquardt nonlinear least squares

Other Useful Packages

  • shiny — Interactive web applications for PK/PD
  • scales — Axis scaling utilities
  • MASS — Various statistical functions

Installation

# R Package Reference {#sec-appendix-packages .unnumbered}

This appendix lists the key R packages used throughout the book, organized by purpose.

## Core PK/PD Packages

- **`rxode2`** — ODE-based PK/PD simulation and modeling engine
- **`nlmixr2`** — Nonlinear mixed-effects modeling (population PK/PD) with FOCEi and SAEM
- **`mrgsolve`** — Fast ODE-based PK/PD simulation in C++
- **`PKNCA`** — Noncompartmental analysis (AUC, Cmax, t½, etc.)
- **`PK`** — Basic pharmacokinetic analysis functions

## Statistical Modeling

- **`nlme`** — Linear and nonlinear mixed-effects models
- **`lme4`** — Linear mixed-effects models
- **`saemix`** — SAEM algorithm for nonlinear mixed-effects models
- **`brms`** — Bayesian regression models using Stan
- **`cmdstanr`** — Lightweight R interface to CmdStan

## Data Manipulation and Visualization

- **`ggplot2`** — Grammar of graphics for publication-quality plots
- **`dplyr`** — Data manipulation (filter, mutate, summarize)
- **`tidyr`** — Tidy data tools (pivot, nest, fill)
- **`readr`** — Fast CSV/file reading
- **`readxl`** — Read Excel files
- **`patchwork`** — Combine multiple ggplot2 plots
- **`ggrepel`** — Non-overlapping text labels in plots

## Reporting and Reproducibility

- **`knitr`** — Dynamic report generation
- **`rmarkdown`** — R Markdown documents
- **`quarto`** — Next-generation publishing system

## Survival and Clinical Endpoints

- **`survival`** — Survival analysis (Kaplan-Meier, Cox PH)
- **`survminer`** — Visualization of survival data

## Differential Equation Solvers

- **`deSolve`** — General ODE/PDE/DDE solver
- **`minpack.lm`** — Levenberg-Marquardt nonlinear least squares

## Other Useful Packages

- **`shiny`** — Interactive web applications for PK/PD
- **`scales`** — Axis scaling utilities
- **`MASS`** — Various statistical functions

## Installation

```r
install.packages(c(
  "rxode2", "nlmixr2", "mrgsolve", "PKNCA", "PK",
  "nlme", "lme4", "saemix", "brms", "cmdstanr",
  "ggplot2", "dplyr", "tidyr", "readr", "readxl",
  "patchwork", "ggrepel",
  "knitr", "rmarkdown", "quarto",
  "survival", "survminer",
  "deSolve", "minpack.lm",
  "shiny", "scales", "MASS"
))
```
