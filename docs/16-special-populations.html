16.1 Learning Objectives

By the end of this chapter you will be able to:

  • Identify PK/PD changes in special populations (pediatrics, geriatrics, pregnancy, renal/hepatic impairment)
  • Model the impact of disease states on drug disposition and effect
  • Analyze drug-drug interaction (DDI) effects on PK parameters
  • Incorporate pharmacogenomic covariates into PK/PD models
  • Handle messy, real-world clinical data with missing doses and uncertain sampling times

16.2 Why Special Populations Matter

Standard PK/PD studies are conducted in healthy volunteers or relatively homogeneous patient groups. But real patients are diverse:

  • Age extremes: Neonates and elderly have altered physiology
  • Organ impairment: Liver and kidney disease affect clearance
  • Pregnancy: Physiological changes alter V and CL
  • Obesity: Dosing by total vs ideal vs adjusted body weight
  • Genetics: Polymorphisms in drug-metabolizing enzymes and transporters

16.2.1 The Goal

Understand how patient characteristics modify PK/PD so we can individualize dosing — the right dose for the right patient.

16.3 Renal Impairment

Renal function is commonly assessed by estimated glomerular filtration rate (eGFR) using the CKD-EPI or MDRD equation. For renally cleared drugs:

CL = CL_{typical} \cdot \left(\frac{eGFR}{90}\right)^{\theta_{renal}}

Code 
library(ggplot2)
library(dplyr)

# Simulate effect of renal function on clearance
egfr_seq <- seq(10, 120, by = 1)
CL_typical <- 5.0  # L/h at eGFR = 90

# Different drugs have different dependence on renal function
theta_renal_values <- c(0.2, 0.5, 0.8, 1.0)
names(theta_renal_values) <- c("Low dependence", "Moderate",
                                "High dependence", "Fully renal")

df_renal <- expand.grid(eGFR = egfr_seq, Category = names(theta_renal_values)) %>%
  mutate(
    theta = theta_renal_values[Category],
    CL = CL_typical * (eGFR / 90)^theta,
    CL_fraction = CL / CL_typical * 100
  )

ggplot(df_renal, aes(eGFR, CL_fraction, color = Category)) +
  geom_line(linewidth = 1) +
  geom_vline(xintercept = c(15, 30, 60, 90), linetype = "dashed", alpha = 0.3) +
  annotate("text", x = c(15, 30, 60, 90), y = 5,
           label = c("ESRD", "Severe", "Moderate", "Normal"),
           angle = 90, size = 3, vjust = -0.5) +
  labs(title = "Effect of Renal Function on Drug Clearance",
       x = "eGFR (mL/min/1.73m²)", y = "Clearance (% of typical)",
       color = "Renal Dependence") +
  scale_color_manual(values = c("#D4A843", "#C44536", "#0B6E6B", "#1B365D")) +
  theme_minimal()

16.4 Hepatic Impairment

The Child-Pugh score classifies liver disease severity (A, B, C). Hepatic impairment affects:

  • Metabolic clearance (CYP450 enzyme activity)
  • Protein binding (albumin synthesis)
  • First-pass metabolism (portal-systemic shunting)
  • Biliary excretion

16.5 Pediatric PK

Children are NOT small adults. PK parameters change dramatically with age due to:

  • Maturation of metabolizing enzymes (CYP450 ontogeny)
  • Changing body composition (water:fat ratio)
  • Developing renal function (GFR matures by ~1 year)
  • Differences in protein binding

16.5.1 Allometric Scaling with Maturation

CL = CL_{adult} \cdot \left(\frac{WT}{70}\right)^{0.75} \cdot \frac{PMA^{\gamma}}{PMA^{\gamma} + PMA_{50}^{\gamma}}

where PMA is post-menstrual age and PMA_{50} is the PMA at 50% mature CL.

Code 
# Pediatric CL maturation
pma_seq <- seq(30, 100, by = 1)  # post-menstrual age in weeks
PMA50 <- 48
gamma_mat <- 3.5

maturation <- pma_seq^gamma_mat / (PMA50^gamma_mat + pma_seq^gamma_mat)

df_maturation <- data.frame(PMA = pma_seq, Maturation = maturation)

ggplot(df_maturation, aes(PMA, Maturation)) +
  geom_line(color = "#0B6E6B", linewidth = 1.2) +
  geom_vline(xintercept = 40, linetype = "dashed", alpha = 0.5) +
  annotate("text", x = 40, y = 0.9, label = "Term birth\n(40 weeks)",
           hjust = -0.1, size = 3) +
  labs(title = "Maturation of Drug Clearance with Age",
       subtitle = paste0("PMA50 = ", PMA50, " weeks"),
       x = "Post-Menstrual Age (weeks)", y = "Fraction of Adult Clearance") +
  theme_minimal()

16.6 Geriatric PK

Aging affects PK through:

  • Reduced renal function (GFR declines ~1% per year after 40)
  • Decreased hepatic mass and blood flow
  • Increased body fat → increased V_d for lipophilic drugs
  • Decreased albumin → increased free fraction
  • Polypharmacy → drug-drug interactions

16.7 Obesity and Body Size Descriptors

Descriptor Formula Use
Total body weight (TBW) Measured Standard
Ideal body weight (IBW) 45.5 + 2.3 \times (Ht - 60) (kg) Loading doses of hydrophilic drugs
Adjusted body weight (ABW) IBW + 0.4 \times (TBW - IBW) Aminoglycosides
Lean body weight (LBW) More complex formulas Most accurate for CL scaling

# Compare body size descriptors
height_in <- 65  # inches

ABW <- IBW + 0.4 * (weight_kg - IBW)

df_body <- data.frame(

16.8 Drug-Drug Interactions (DDI)

DDIs can be classified as:

Mechanism Effect on Victim Drug Example
CYP inhibition ↓ CL → ↑ AUC Ketoconazole + midazolam
CYP induction ↑ CL → ↓ AUC Rifampicin + warfarin
Transporter inhibition ↓ renal/biliary CL Probenecid + penicillin
Protein binding displacement ↑ free fraction (Rarely clinically significant)
pH alteration ± absorption Antacids + ketoconazole
Code 
# Simulate DDI: Competitive CYP inhibition
dose <- 100  # mg
V <- 50      # L

# Without inhibitor
CL_base <- 5.0
ke_base <- CL_base / V

# With inhibitor (competitive, increases Km_apparent)
I <- 2.0       # inhibitor concentration (mg/L)
Ki <- 0.5      # inhibition constant (mg/L)
fm <- 0.8      # fraction metabolized by inhibited pathway

CL_inhibited <- CL_base * (1 - fm + fm / (1 + I/Ki))

time <- seq(0, 24, by = 0.1)
conc_base <- (dose/V) * exp(-ke_base * time)
conc_inh   <- (dose/V) * exp(-(CL_inhibited/V) * time)

df_ddi <- data.frame(
  time = rep(time, 2),
  conc = c(conc_base, conc_inh),
  Scenario = rep(c("Without Inhibitor", "With Inhibitor"), each = length(time))
)

ggplot(df_ddi, aes(time, conc, color = Scenario)) +
  geom_line(linewidth = 1) +
  labs(title = "Drug-Drug Interaction: CYP Inhibition",
       subtitle = paste0("CL reduced from ", CL_base, " to ", round(CL_inhibited, 1), " L/h"),
       x = "Time (hours)", y = "Concentration (mg/L)") +
  scale_color_manual(values = c("#0B6E6B", "#C44536")) +
  theme_minimal()

16.8.1 Enzyme Induction

While inhibition reduces CL, enzyme induction increases it over time by upregulating metabolic enzymes (via PXR/CAR nuclear receptors). The enzyme level E(t) evolves slowly:

\frac{dE}{dt} = k_{syn} \cdot \left(1 + \frac{I_{max} \cdot C(t)}{IC_{50} + C(t)}\right) - k_{deg} \cdot E

At steady state: E_{ss} = \frac{k_{syn}}{k_{deg}} \cdot \left(1 + \frac{I_{max} \cdot C_{ss}}{IC_{50} + C_{ss}}\right)

Autoinduction occurs when a drug induces its own metabolism — CL rises and steady-state concentrations decline over days to weeks. This is the classic pattern for rifampicin, carbamazepine, and St. John’s Wort.

Code 
# Simulate autoinduction
library(rxode2)
library(ggplot2)
library(dplyr)

mod_autoind <- rxode2({
  # Clearance depends on enzyme level
  CL <- CL_base * enzyme
  d/dt(depot)   <- -ka * depot
  d/dt(central) <-  ka * depot - (CL/V) * central
  # Enzyme level: induction by drug concentration
  d/dt(enzyme)  <- ksyn * (1 + Imax * central/V / (IC50 + central/V)) - kdeg * enzyme
  conc <- central / V
})

params_ai <- c(ka = 1.2, V = 62, CL_base = 7.5,
               ksyn = 0.04, kdeg = 0.04, Imax = 3, IC50 = 3)

# Initial conditions: enzyme(0) = 1 (baseline = ksyn/kdeg)
inits_ai <- c(depot = 0, central = 0, enzyme = 1)

times_ai <- seq(0, 336, by = 1)  # 14 days
event_ai <- et(time = seq(0, 336, 24), amt = rep(400, 15))

sim_ai <- rxSolve(mod_autoind, params_ai, events = event_ai %>% et(times_ai), 
                   inits = inits_ai)
df_ai <- as.data.frame(sim_ai)

ggplot(df_ai, aes(time)) +
  geom_line(aes(y = conc, color = "Concentration"), linewidth = 0.8) +
  geom_line(aes(y = enzyme * 5, color = "Enzyme Level (×5)"), linewidth = 1) +
  labs(title = "Autoinduction: CL Increases Over Time",
       subtitle = "Steady-state concentrations decline as enzyme is induced",
       x = "Time (hours)", y = "Value", color = "") +
  scale_color_manual(values = c("#C44536", "#0B6E6B")) +
  theme_minimal()

16.8.2 DDI PD Interactions: Synergy, Antagonism, and Bliss Independence

When two drugs interact pharmacodynamically (not just PK), we need models for their combined effect.

Bliss Independence (the no-interaction reference):

E_{AB} = E_A + E_B - E_A \cdot E_B

where effects are expressed as fractions (0–1).

Synergy (combined effect exceeds expectation):

E_{AB} = E_{Bliss} + \gamma \cdot E_{Bliss} \cdot (1 - E_{Bliss}), \quad \gamma > 0

Antagonism (combined effect below expectation):

E_{AB} = E_A \cdot E_B \quad \text{(multiplicative dampening)}

Observation vs Bliss Classification
E_{AB} > Bliss prediction Synergy (\gamma > 0)
E_{AB} < Bliss prediction Antagonism
E_{AB} \approx Bliss prediction No interaction

16.8.3 Combined Clearance: Mechanistic vs Empirical

When DDI data includes enzyme-kinetic parameters (K_i):

CL_A = \frac{V_{max,A}}{K_{m,A} \cdot (1 + C_B/K_i) + C_A}

Without mechanistic data, fall back to additive (CL_A + CL_B) or multiplicative forms — but treat these as approximations, not mechanism.

16.9 Pharmacogenomics

Genetic polymorphisms in drug-metabolizing enzymes and transporters can profoundly affect PK:

Gene Phenotype PK Effect
CYP2D6 Poor metabolizer (PM) ↓ CL, ↑ AUC (5–10×)
CYP2C19 PM / IM / EM / UM Variable CL
TPMT Low activity Risk of myelosuppression with thiopurines
UGT1A1 Reduced activity ↑ irinotecan toxicity
SLCO1B1 Reduced transport ↑ statin exposure

Real-world PK data is rarely clean. Common issues and solutions:

# Example: messy PK data
messy_data <- data.frame(

  stringsAsFactors = FALSE

Code 
# Clean it up
clean_data <- messy_data %>%
  # Convert ID to numeric (NA for non-numeric)
  mutate(ID = as.numeric(ID)) %>%
  # Filter out rows with no ID
  filter(!is.na(ID)) %>%
  # Clean time: remove non-numeric characters
  mutate(TIME = as.numeric(gsub("[^0-9.]", "", TIME))) %>%
  # Handle BLOQ (set to LLOQ/2 = 0.05)
  mutate(CONC = case_when(
    CONC == "BLQ" ~ 0.05,
    grepl(">", CONC) ~ as.numeric(gsub(">", "", CONC)),
    TRUE ~ as.numeric(CONC)
  )) %>%
  # Forward-fill dose within subject
  mutate(DOSE = as.numeric(gsub("mg", "", DOSE))) %>%
  group_by(ID) %>%
  tidyr::fill(DOSE, .direction = "down") %>%
  ungroup()

cat("\nCleaned data:\n")

16.10.1 Exercises

  1. Adjust a maintenance dose for a patient with eGFR = 30 mL/min. The drug is 80% renally cleared with \theta_{renal} = 0.7, typical CL = 10 L/h.
  2. Compute the dose for an obese patient (TBW = 150 kg, height = 170 cm) using IBW, ABW, and LBW. Compare the results.
  3. Simulate the PK profile of a CYP2D6 substrate in a poor metabolizer (CL reduced by 80%) vs an extensive metabolizer.
  4. Write R code to detect and flag implausible concentration values (e.g., increasing when they should be decreasing).

16.10.2 Common Mistakes

  • Using adult dosing in children without allometric scaling
  • Ignoring the free drug hypothesis — protein binding changes in disease states
  • Using Cockcroft-Gault for eGFR in unstable renal function — it assumes steady state
  • Assuming DDI magnitude from in vitro data — always verify with clinical data
  • Using TBW for dosing hydrophilic drugs in obesity — may overdose

16.10.3 Key Takeaway

Patient factors — age, organ function, body size, genetics, and concomitant medications — profoundly affect PK/PD. Dose individualization requires quantifying these effects through covariate modeling. Always consider the patient before the population average.

## Drug-Drug Interactions (DDI)

DDIs can be classified as:

| Mechanism | Effect on Victim Drug | Example |
|-----------|----------------------|---------|
| **CYP inhibition** | ↓ CL → ↑ AUC | Ketoconazole + midazolam |
| **CYP induction** | ↑ CL → ↓ AUC | Rifampicin + warfarin |
| **Transporter inhibition** | ↓ renal/biliary CL | Probenecid + penicillin |
| **Protein binding displacement** | ↑ free fraction | (Rarely clinically significant) |
| **pH alteration** | ± absorption | Antacids + ketoconazole |

```{r}
# Simulate DDI: Competitive CYP inhibition
dose <- 100  # mg
V <- 50      # L

# Without inhibitor
CL_base <- 5.0
ke_base <- CL_base / V

# With inhibitor (competitive, increases Km_apparent)
I <- 2.0       # inhibitor concentration (mg/L)
Ki <- 0.5      # inhibition constant (mg/L)
fm <- 0.8      # fraction metabolized by inhibited pathway

CL_inhibited <- CL_base * (1 - fm + fm / (1 + I/Ki))

time <- seq(0, 24, by = 0.1)
conc_base <- (dose/V) * exp(-ke_base * time)
conc_inh   <- (dose/V) * exp(-(CL_inhibited/V) * time)

df_ddi <- data.frame(
  time = rep(time, 2),
  conc = c(conc_base, conc_inh),
  Scenario = rep(c("Without Inhibitor", "With Inhibitor"), each = length(time))
)

ggplot(df_ddi, aes(time, conc, color = Scenario)) +
  geom_line(linewidth = 1) +
  labs(title = "Drug-Drug Interaction: CYP Inhibition",
       subtitle = paste0("CL reduced from ", CL_base, " to ", round(CL_inhibited, 1), " L/h"),
       x = "Time (hours)", y = "Concentration (mg/L)") +
  scale_color_manual(values = c("#0B6E6B", "#C44536")) +
  theme_minimal()
```

### Enzyme Induction

While inhibition reduces CL, **enzyme induction** increases it over time by upregulating metabolic enzymes (via PXR/CAR nuclear receptors). The enzyme level $E(t)$ evolves slowly:

$$\frac{dE}{dt} = k_{syn} \cdot \left(1 + \frac{I_{max} \cdot C(t)}{IC_{50} + C(t)}\right) - k_{deg} \cdot E$$

At steady state: $E_{ss} = \frac{k_{syn}}{k_{deg}} \cdot \left(1 + \frac{I_{max} \cdot C_{ss}}{IC_{50} + C_{ss}}\right)$

**Autoinduction** occurs when a drug induces its own metabolism — CL rises and steady-state concentrations decline over days to weeks. This is the classic pattern for rifampicin, carbamazepine, and St. John's Wort.

```{r}
# Simulate autoinduction
library(rxode2)
library(ggplot2)
library(dplyr)

mod_autoind <- rxode2({
  # Clearance depends on enzyme level
  CL <- CL_base * enzyme
  d/dt(depot)   <- -ka * depot
  d/dt(central) <-  ka * depot - (CL/V) * central
  # Enzyme level: induction by drug concentration
  d/dt(enzyme)  <- ksyn * (1 + Imax * central/V / (IC50 + central/V)) - kdeg * enzyme
  conc <- central / V
})

params_ai <- c(ka = 1.2, V = 62, CL_base = 7.5,
               ksyn = 0.04, kdeg = 0.04, Imax = 3, IC50 = 3)

# Initial conditions: enzyme(0) = 1 (baseline = ksyn/kdeg)
inits_ai <- c(depot = 0, central = 0, enzyme = 1)

times_ai <- seq(0, 336, by = 1)  # 14 days
event_ai <- et(time = seq(0, 336, 24), amt = rep(400, 15))

sim_ai <- rxSolve(mod_autoind, params_ai, events = event_ai %>% et(times_ai), 
                   inits = inits_ai)
df_ai <- as.data.frame(sim_ai)

ggplot(df_ai, aes(time)) +
  geom_line(aes(y = conc, color = "Concentration"), linewidth = 0.8) +
  geom_line(aes(y = enzyme * 5, color = "Enzyme Level (×5)"), linewidth = 1) +
  labs(title = "Autoinduction: CL Increases Over Time",
       subtitle = "Steady-state concentrations decline as enzyme is induced",
       x = "Time (hours)", y = "Value", color = "") +
  scale_color_manual(values = c("#C44536", "#0B6E6B")) +
  theme_minimal()
```

### DDI PD Interactions: Synergy, Antagonism, and Bliss Independence

When two drugs interact pharmacodynamically (not just PK), we need models for their combined effect.

**Bliss Independence** (the no-interaction reference):

$$E_{AB} = E_A + E_B - E_A \cdot E_B$$

where effects are expressed as fractions (0–1).

**Synergy** (combined effect exceeds expectation):

$$E_{AB} = E_{Bliss} + \gamma \cdot E_{Bliss} \cdot (1 - E_{Bliss}), \quad \gamma > 0$$

**Antagonism** (combined effect below expectation):

$$E_{AB} = E_A \cdot E_B \quad \text{(multiplicative dampening)}$$

| Observation vs Bliss | Classification |
|---|---|
| $E_{AB} >$ Bliss prediction | Synergy ($\gamma > 0$) |
| $E_{AB} <$ Bliss prediction | Antagonism |
| $E_{AB} \approx$ Bliss prediction | No interaction |

### Combined Clearance: Mechanistic vs Empirical

When DDI data includes enzyme-kinetic parameters ($K_i$):

$$CL_A = \frac{V_{max,A}}{K_{m,A} \cdot (1 + C_B/K_i) + C_A}$$

Without mechanistic data, fall back to additive ($CL_A + CL_B$) or multiplicative forms — but treat these as approximations, not mechanism.

## Handling Messy Clinical Data

Real-world PK data is rarely clean. Common issues and solutions:

```{r}
# Handling common data quality issues
library(dplyr)

# Example: messy PK data
messy_data <- data.frame(
  ID = c(1, 1, 1, "2", "2", "2", 3, 3, NA_character_),
  TIME = c(0, 2, "MISSING", 0, 1, 4, "PRE", "2h", 6),
  CONC = c("BLQ", 5.2, 2.1, 0, ">100", 8.5, NA, 3.7, 1.2),
  DOSE = c("100mg", "", "", "200mg", "", "", "50 mg", "", ""),
  stringsAsFactors = FALSE
)

cat("Raw messy data:\n")
print(messy_data)

# Clean it up
clean_data <- messy_data %>%
  # Convert ID to numeric (NA for non-numeric)
  mutate(ID = as.numeric(ID)) %>%
  # Filter out rows with no ID
  filter(!is.na(ID)) %>%
  # Clean time: remove non-numeric characters
  mutate(TIME = as.numeric(gsub("[^0-9.]", "", TIME))) %>%
  # Handle BLOQ (set to LLOQ/2 = 0.05)
  mutate(CONC = case_when(
    CONC == "BLQ" ~ 0.05,
    grepl(">", CONC) ~ as.numeric(gsub(">", "", CONC)),
    TRUE ~ as.numeric(CONC)
  )) %>%
  # Forward-fill dose within subject
  mutate(DOSE = as.numeric(gsub("mg", "", DOSE))) %>%
  group_by(ID) %>%
  tidyr::fill(DOSE, .direction = "down") %>%
  ungroup()

cat("\nCleaned data:\n")
print(clean_data)
```

::: {.mistakes-box}
### Common Mistakes

- **Using adult dosing in children without allometric scaling**
- **Ignoring the free drug hypothesis** — protein binding changes in disease states
- **Using Cockcroft-Gault for eGFR in unstable renal function** — it assumes steady state
- **Assuming DDI magnitude from in vitro data** — always verify with clinical data
- **Using TBW for dosing hydrophilic drugs in obesity** — may overdose
:::

::: {.key-box}
### Key Takeaway

Patient factors — age, organ function, body size, genetics, and concomitant medications — profoundly affect PK/PD. Dose individualization requires quantifying these effects through covariate modeling. Always consider the patient before the population average.
:::
