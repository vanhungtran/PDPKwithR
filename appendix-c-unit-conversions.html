## Common PK Unit Conversions in R

```{r}
# Useful conversion functions
mg_to_ug <- function(x) x * 1000
ug_to_ng <- function(x) x * 1000
L_to_mL <- function(x) x * 1000
hour_to_min <- function(x) x * 60
min_to_hour <- function(x) x / 60
CL_Lh_to_mLmin <- function(x) x * 1000 / 60
CL_mLmin_to_Lh <- function(x) x * 60 / 1000

# Example
CL_mLmin_to_Lh(120)  # 120 mL/min = 7.2 L/h
```

## Dose Normalization

- mg/kg → mg: multiply by body weight (kg)
- mg/m² → mg: multiply by body surface area (m²)

$$BSA (m^2) = \sqrt{\frac{Height(cm) \times Weight(kg)}{3600}}$$

## Temperature

- °C to K: add 273.15
- K to °C: subtract 273.15
