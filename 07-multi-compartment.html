7.2 Why Multi-Compartment Models?

After IV bolus dosing, many drugs show a biphasic decline in concentration: a rapid initial drop (distribution phase) followed by a slower decline (elimination phase). A one-compartment model cannot capture this behavior.

Code 
library(rxode2)
library(dplyr)
library(ggplot2)

# Compare one- vs two-compartment profiles
times <- seq(0, 24, by = 0.05)

# One-compartment
mod_1cmt <- rxode2({
  d/dt(central) <- -ke * central
  conc <- central / V
})

# Two-compartment
mod_2cmt <- rxode2({
  d/dt(central)    <- -k10 * central - k12 * central + k21 * peripheral
  d/dt(peripheral) <-  k12 * central - k21 * peripheral
  conc <- central / V1
})

params_1 <- c(ke = 0.3, V = 15)
params_2 <- c(k10 = 0.3, k12 = 1.5, k21 = 0.8, V1 = 5)

event <- et(time = 0, amt = 100)

sim1 <- rxSolve(mod_1cmt, params_1, event %>% et(times))
sim2 <- rxSolve(mod_2cmt, params_2, event %>% et(times))

df <- bind_rows(
  as.data.frame(sim1) %>% mutate(Model = "One-Compartment"),
  as.data.frame(sim2) %>% mutate(Model = "Two-Compartment")
)

ggplot(df, aes(time, conc, color = Model)) +
  geom_line(linewidth = 1) +
  scale_y_log10() +
  labs(title = "One- vs Two-Compartment Models",
       subtitle = "Note the biphasic decline in the two-compartment model",
       x = "Time (hours)", y = "Concentration (mg/L, log scale)") +
  scale_color_manual(values = c("#0B6E6B", "#C44536")) +
  theme_minimal()

7.3 Two-Compartment Model Structure

7.3.1 Model Equations

\frac{dA_1}{dt} = -k_{10}A_1 - k_{12}A_1 + k_{21}A_2 \frac{dA_2}{dt} = k_{12}A_1 - k_{21}A_2 C_1(t) = \frac{A_1(t)}{V_1}

where: - A_1, A_2: amounts in central and peripheral compartments - V_1: volume of central compartment - k_{10}: elimination rate constant from central compartment - k_{12}: transfer rate from central to peripheral - k_{21}: transfer rate from peripheral to central

7.3.2 Analytical Solution

C(t) = A \cdot e^{-\alpha t} + B \cdot e^{-\beta t}

where \alpha (distribution) and \beta (elimination) are hybrid rate constants (\alpha > \beta), and A and B are hybrid coefficients.

Code 
# Visualize the analytical solution components
alpha <- 2.8
beta  <- 0.25
A_coef <- 12
B_coef <- 8

times <- seq(0, 24, by = 0.05)
C_distribution <- A_coef * exp(-alpha * times)
C_elimination  <- B_coef * exp(-beta * times)
C_total <- C_distribution + C_elimination

df_comp <- data.frame(
  time = rep(times, 3),
  conc = c(C_total, C_distribution, C_elimination),
  Component = rep(c("Total", "Distribution (α)", "Elimination (β)"), each = length(times))
)

ggplot(df_comp, aes(time, conc, color = Component, linetype = Component)) +
  geom_line(linewidth = 1) +
  scale_y_log10() +
  labs(title = "Two-Compartment Model: Analytical Components",
       subtitle = paste0("C(t) = ", round(A_coef, 1), "·e^{-", alpha, "t} + ",
                         round(B_coef, 1), "·e^{-", beta, "t}"),
       x = "Time (hours)", y = "Concentration (mg/L, log scale)") +
  scale_color_manual(values = c("#0B6E6B", "#D4A843", "#C44536")) +
  theme_minimal()

7.4 Micro- and Macro-Constants

Parameter Type Symbols Meaning
Micro-constants k_{10}, k_{12}, k_{21} Rate constants between compartments
Macro-constants A, B, \alpha, \beta Hybrid coefficients and exponents

7.4.1 Conversion Between Micro and Macro

\alpha + \beta = k_{10} + k_{12} + k_{21} \alpha \cdot \beta = k_{10} \cdot k_{21} A = \frac{D(\alpha - k_{21})}{V_1(\alpha - \beta)}, \quad B = \frac{D(k_{21} - \beta)}{V_1(\alpha - \beta)}

Code 
# Convert micro to macro constants
micro_to_macro <- function(k10, k12, k21, V1, dose) {
  sum_k <- k10 + k12 + k21
  prod_k <- k10 * k21

  # Alpha and beta are roots of: r^2 - sum_k * r + prod_k = 0
  disc <- sqrt(sum_k^2 - 4 * prod_k)
  alpha <- (sum_k + disc) / 2
  beta  <- (sum_k - disc) / 2

  A <- dose * (alpha - k21) / (V1 * (alpha - beta))
  B <- dose * (k21 - beta) / (V1 * (alpha - beta))

  c(alpha = alpha, beta = beta, A = A, B = B)
}

macro <- micro_to_macro(k10 = 0.3, k12 = 1.5, k21 = 0.8, V1 = 5, dose = 100)

7.5 Fitting a Two-Compartment Model

7.5.1 Using nls() with the Analytical Solution

Code 
# Simulate noisy two-compartment data
set.seed(42)
sample_times <- c(0.05, 0.1, 0.25, 0.5, 1, 2, 4, 6, 8, 12, 24)
true_macro <- micro_to_macro(0.3, 1.5, 0.8, 5, 100)

true_conc <- true_macro["A"] * exp(-true_macro["alpha"] * sample_times) +
             true_macro["B"] * exp(-true_macro["beta"] * sample_times)

obs_2cmt <- data.frame(
  time = sample_times,
  conc = true_conc * rlnorm(length(sample_times), 0, 0.1)
)

# Fit two-compartment model
fit_2cmt <- nls(conc ~ A * exp(-alpha * time) + B * exp(-beta * time),
                data = obs_2cmt,
                start = list(A = 15, alpha = 2.0, B = 10, beta = 0.2),
                lower = c(0, 0, 0, 0),
                algorithm = "port")

7.7 Identifiability and Parameter Interpretation

7.7.1 When One-Compartment Models Are Not Enough

  • Distribution phase is clinically relevant (e.g., rapid onset drugs)
  • Drug shows clear biphasic decline on semi-log plot
  • Peripheral compartment has physiological meaning (e.g., tissue binding)
  • Accurate prediction of early time points is needed

7.7.2 Common Issues

  • Parameter identifiability: With sparse data, 4 parameters (A, B, \alpha, \beta) may not be uniquely estimable
  • Correlation: A and \alpha are often highly correlated
  • Initial estimates: Good starting values are critical; use graphical methods (method of residuals)
Code 
# Method of residuals for initial estimates
# Step 1: Fit terminal phase to get B and beta
terminal_idx <- 6:length(sample_times)  # Choose terminal points
lm_term <- lm(log(obs_2cmt$conc[terminal_idx]) ~ obs_2cmt$time[terminal_idx])
B_init <- exp(coef(lm_term)[1])
beta_init <- -coef(lm_term)[2]

# Step 2: Calculate residuals and fit distribution phase
terminal_pred <- B_init * exp(-beta_init * sample_times)
residuals <- obs_2cmt$conc - terminal_pred
residuals <- residuals[residuals > 0]

cat("Initial estimates from method of residuals:\n")

7.7.4 Common Mistakes

  • Over-parameterization: A two-compartment model with sparse data may fit noise, not signal
  • Misidentifying phases: The initial rapid decline may be due to distribution, not elimination
  • Using one-compartment model when distribution matters: e.g., for drugs with rapid onset of action
  • Poor initial estimates: nls() is sensitive to starting values; use method of residuals

7.7.5 Key Takeaway

Multi-compartment models capture the distribution phase that one-compartment models miss. The two-compartment model with parameters (A, B, \alpha, \beta) or (k_{10}, k_{12}, k_{21}, V_1) is the most common structural model in PK. Always check semi-log plots for biphasic behavior before choosing your model.

# Simulate noisy two-compartment data
set.seed(42)
